Code Name Elisabeth

Code Name Elisabeth is a Marit Larsen fan-site based in China. A Marit Larsen fansite aims to provide news updates and fans interaction.

The site name "Code Name Elisabeth" origins from a short interview video with Marit Larsen, in which Marit is asked to pick a name that she prefers from her given name Marit and her middle name Elisabeth. Marit picks her middle name for it is her secret name.

With the idea explained, the prefix "Code Name" is designed to create an atomsphere of mystery, profession and project-related context.

The site now has three sub sites, namely Marit Larsen News(in Chinese), Marit Larsen Forum(in Chinese) and Marit Larsen Lyrics(in English).

By the way, the site description "The Least Popular Marit Larsen Fan-site Based in China" is half a joke.
